# 文件
## 常用命令 
### :ls 命令
#### 功能
    查看缓冲区列表
#### 输出示例:
~~~
:ls
  1 %a   "a.txt"                        第 16 行
  2 #    "b.txt"                        第 1 行

  %符号表示哪个缓冲区在当前窗口可见
  #符号代表轮换文件
  ctrl-^ 可以在当前文件和轮换文件快速切换
  可以用四个命令遍历缓冲区列表
    :bn[ext] 正向移动列表
    :bp[rev] 反向移动列表
    :bf[irst] 跳到列表的开头
    :bl[ast] 跳到列表的结尾
~~~
